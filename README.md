# Static Landing Pages

This project is plain HTML, CSS, and JavaScript.

## Structure

```text
lp/
  admissions/
    index.html
    thankyou.html
    Images/
    Scripts/
    style/
```

Add new landing pages by creating another folder under `lp`, for example:

```text
lp/commerce/index.html
lp/commerce/thankyou.html
```

## Commands

```sh
npm run dev
npm run build
npm run preview
```

`npm run build` copies `lp` to `dist/lp`.
